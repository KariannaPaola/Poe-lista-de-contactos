Contact List App

Aplicación web que muestra una lista de contactos y permite buscar contactos por nombre en tiempo real.
La lista de contactos está definida en el código y se filtra dinámicamente según lo que el usuario escriba en el buscador.

Live Demo
https://kariannapaola.github.io/Poe-lista-de-contactos/

Tech Stack
JavaScript
Tailwind CSS
HTML

Features
Buscador que filtra contactos por nombre.
Interfaz simple e intuitiva.
Renderizado dinámico de la lista de contactos.


What I Learned
Manipulación del DOM con JavaScript.
Implementación de filtros dinámicos en listas.
Manejo de eventos de entrada (input events).
Uso de Tailwind CSS para construir interfaces simples y responsivas.
